Circuit Design

Exploring the electronics that power our LED Resin Cube

Electronics Overview

The LED Resin Cube relies on carefully designed circuitry to control the LED matrix and process input from the IR sensor. Below you'll find detailed information about our circuit designs, component choices, and electronic systems.

Hover over the circuit diagrams and images to see them in greater detail.

Main Control Board

Main Control Board

The primary control board houses the microcontroller that serves as the brain of our LED cube. This circuit provides the following functionalities:

  • Power regulation for all system components
  • LED matrix control via dedicated drivers
  • IR signal reception and processing
  • Programming interface for firmware updates

The board uses a 32-bit ARM microcontroller running at 120MHz, providing ample processing power for complex LED animations and fast response to IR inputs.

LED Matrix Design

LED Matrix Design

The 3D LED matrix forms the visual heart of our project. Each LED is individually addressable, allowing for complex 3D animations and patterns.

  • 5x5x5 grid configuration (125 LEDs total)
  • WS2812B individually addressable RGB LEDs
  • Serial data connection with optimized signal routing
  • Carefully designed power distribution to prevent voltage drop

The matrix design presented significant challenges in signal integrity and power distribution due to the 3D arrangement in resin. We developed a custom routing solution to ensure reliable data transmission to each LED.

IR Sensor System

The IR sensor system detects and interprets signals from the IR wand controller. This sub-circuit includes:

  • TSOP38238 IR receiver with built-in filtering
  • Signal conditioning circuitry for noise rejection
  • Digital interface to the main microcontroller
  • 38kHz carrier frequency detection

The sensor has a 120-degree detection angle, allowing for interaction with the cube from various positions. Signal processing algorithms in the microcontroller interpret different movement patterns from the IR wand.

Power Supply System

Power Supply System

Reliable power is critical for our LED cube to function properly. Our power supply system incorporates:

  • 5V/3A main power supply for LEDs
  • 3.3V regulated supply for logic circuits
  • Power filtering and protection circuitry
  • Capacitive buffering for LED current spikes

The power system is designed to handle the varying current demands of the LED matrix during different animation patterns. High-capacity filter capacitors ensure stable operation even when all LEDs are at maximum brightness.

IR Wand Controller

IR Wand Controller

The IR wand controller is the user interface for our LED cube. This compact circuit includes:

  • ATtiny85 microcontroller for command generation
  • IR LED with driver circuit
  • Motion detection via ADXL345 accelerometer
  • Button inputs for mode selection
  • Battery power system with power management

The wand translates physical movements and button presses into IR command signals that are sent to the cube. Different gestures trigger different light patterns and effects in the LED matrix.

Key Components

Component Specifications Purpose
Microcontroller ARM Cortex-M4, 120MHz, 256KB Flash Main control system, LED pattern generation, IR signal processing
WS2812B LEDs 5x5x5 Matrix, 5V operation, 60mA max per LED Visual display elements, individually addressable RGB lights
TSOP38238 38kHz IR receiver, 45m range Detects and demodulates IR signals from the wand controller
ATtiny85 8MHz, 8KB Flash, 512B RAM Controls the IR wand, processes accelerometer data
ADXL345 3-axis accelerometer, ±16g range Motion detection for the IR wand controller
Power Regulator 5V to 3.3V buck converter, 1A output Provides stable power for logic circuits
Capacitors 1000μF, 10V electrolytic, various ceramics Power filtering and stabilization for LEDs
