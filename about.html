About the Project

Project Overview

The LED Resin Cube is a capstone engineering project that combines electronics, programming, and materials science to create an interactive light display. The cube features embedded LEDs within a clear resin structure that respond dynamically to user input via an IR wand controller.

This project demonstrates the practical application of concepts from circuit design, microcontroller programming, sensor integration, and material properties of resins and LEDs.

Key Features

Interactive Design

The cube responds in real-time to user movements and commands from the IR wand.

Dynamic Lighting

Multiple LED patterns and animations create stunning visual displays.

Custom Circuitry

Carefully designed PCB ensures reliable performance in a compact form factor.

Advanced Programming

Efficient code handles input processing and controls complex lighting patterns.

Project Goals

This project was developed with the following goals in mind:

  • Create an engaging interactive experience through LED lighting patterns
  • Demonstrate practical integration of electronics within a resin medium
  • Develop a custom controller system using IR technology
  • Implement efficient programming to create responsive interactions
  • Design a visually appealing product with practical applications

Technologies Used

Hardware

  • Addressable RGB LEDs
  • Custom PCB Design
  • IR Sensors and Emitters
  • Microcontroller
  • Clear Casting Resin

Software

  • C/C++ Programming
  • LED Animation Libraries
  • IR Communication Protocols
  • PCB Design Software
